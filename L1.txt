Qualquer coisinha aí.

Git init == Inicia um repositório local do Git
Git status == Exibe status do Git 
Git add . (Ou o nome da pasta/aquivo) == Adiciona no repositório local do Git
Git commit -m "Mensagem de alteração" == Adiciona um check point 
Git log == Mostra histórico 
Git checkout == localizar ou trocar versão do arquivo
Git merge == Mescla linhas temporais
Git reset --soft HEAD~1 == Remove o último commit
Git branch -M main == Troca o nome de master para main
